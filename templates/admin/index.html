Introduction

This is introduction how to use this backend.
Categories
At first you should add categories (example Tvorim web, Programovani, Data girls, etc..). Categories you can add in Courses/Categories section.
Venues
In the next step create venues for all cities. Then you can add venue to the courses. Venues you can add in Venues/Course venues section.
Admins/Organisers
You can add more administrators of the system, who can manage courses backend. New administrators you can add in Auth/Users section. You should setup password for the organiser and then send by email his credentials. Mandatory fields are email and password.
When you create new admin user, you are redirect to the next screen, where you should check Staff status, Superuser status and save the user.
Couches
You can also add new couches. You add new couch in Auth/User section. There is only one mandatory field email address. You do not need fill password for the couch. At the final step, you check couch . After this step , you will be redirect to the next page, where is no need to setup any other information if you do not need and save the user.
